# linshigong skill

linshigong 是一个问题解决方法的指导框架。当 AI 解决问题时，它要求遵循"临时工思维"——先问自己还有什么没想到的，主动发现同类问题并闭环，穷尽一切方法后再放弃。每一句话都嵌入底层逻辑、闭环、颗粒度等关键词，用结果说话、用数据验证。

## 用法

拉取代码
```
git clone gitUrl
```

## 创建链接
创建链接到具体工程的 skills 目录，或者创建链接到 ai ide 工具的全局目录
windows下
```
mklink /D "linshigong" "你的路径\ai-skills\linshigong"
```

## 命令
```
/linshigong
```
